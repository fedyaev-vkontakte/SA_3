# SA_3

## Анализ удержания пользователей и когортный анализ активности в социальном сервисе

## Описание проекта

### Цель
Провести комплексный анализ пользовательской активности для оценки удержания аудитории (retention), сравнения эффективности разных источников трафика (платный vs органический) и выявления паттернов оттока пользователей.

### Задачи
1. Исследовать демографический портрет аудитории — распределение по полу, возрасту, городам

2. Проанализировать временные паттерны активности — почасовая, дневная и недельная динамика просмотров и лайков

3. Построить когортные таблицы для визуализации удержания пользователей во времени

4. Сравнить retention платного и органического трафика для оценки окупаемости рекламных кампаний

5. Рассчитать ключевую метрику Day 20 Retention для 30-й по счету когорты

6. Детализировать когортный анализ по странам и ОС — выявление сегментов с лучшим/худшим удержанием

7. Провести недельный анализ оттока с классификацией пользователей на new (новые), retained (вернувшиеся) и gone (ушедшие)

### Данные
Источник: Аналитической БД

Основные таблицы:

feed_actions — действия пользователей в ленте (просмотры, лайки, посты)

message_actions — действия в сообщениях (используется в одном из запросов)


### Ключевые выводы, которые можно получить
+ В первый день когорта платных пользователей обычно больше, но в долгосроке они удерживаются хуже органических

+ На 20-й день после первого визита retention составляет определенный процент (расчетный)

+ Есть часы и дни недели с пиковой активностью — зоны для оптимизации нагрузки

+ Некоторые страны/ОС показывают аномально низкое удержание — требуют отдельного анализа
